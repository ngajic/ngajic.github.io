Nv2.0 Highscore Charts

This repository is used for storing files associated with highscoring in Nv2.0 game, along with code necessary for presenting charts.
You wanna be on these charts? Download game from here: Download Nv2.0; And start playing!

GitHub Pages is used for easy Web hosting. Just type the name of this repository and GitHub will do its job! (eg. ngajic.github.io)

I am using Highcharts' .js files to present charts. Code and many examples for it are available at their GitHub account: Highcharts.

I try to update these charts daily. If any problems, contact me over GitHub account or in Discord server dedicated for this awesome platformer franchise.
